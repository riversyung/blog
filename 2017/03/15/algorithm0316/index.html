Divide-and-Conquer分治算法

  • 表现形式

  • 从时间复杂度的角度分析一下

    • Divide阶段
      • 将问题分解成a个子问题
      • 每个自问题的规模为原问题的n/b
      • 划分时间是D(n)
    • Conquer阶段
      • 递归调用（直到子问题足够小）
      • Conquer时间 = aT(n/b)
    • Combine阶段
      • C(n) 合并阶段的时间，一般线性，可以直接得到
    • 这里时间复杂度计算的时候，大多数情况下都可以直接用Master定理求解

  • Master定理小解

    • T(n) = aT(n/b) + f(n), 则T(n)可以如下求解
    • 若f(n) = O(n^(logb^(a-e))) ,e>0是常数，则T(n) = theata(n^(logb^(a)))
    • 若f(n) = theta(n^logb^(a)), 则T(n) = theta(n^(logb^(a)lgn)
    • 若f(n) = omiga(n^logb^(a+e)) ,e>0的常数，则T(n) = theta(f(n))
    • Master定理可以直观理解一下，谁大就趋近于谁

  • 实例：求一个集合中的最大数算法
